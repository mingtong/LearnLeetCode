## 栈

Stack在C#和Java中都有现成的定义，且在LeetCode中也可以直接使用。 
其泛型版本定义的主要属性和方法如下：
``` C#
public class Stack<T>
{
    public int Count;
    public void Clear();
    public T Peek(); //返回的对象顶部的 Stack 而不删除它。
    public T Pop();  //删除并返回 Stack 顶部的对象。
    public void Push(T); //在 Stack 的顶部插入一个对象。
}
```
